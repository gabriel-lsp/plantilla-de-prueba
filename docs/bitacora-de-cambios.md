# Bitácora de cambios

Este documento registra los avances principales del Banco Digital del Sistema Braille.

La bitácora permite dejar evidencia del desarrollo progresivo del proyecto, facilitar la revisión institucional y diferenciar la versión original de futuras mejoras o adaptaciones.

## Versión 1.0 funcional

Se crea el Banco Digital del Sistema Braille como plataforma web educativa para la consulta visual, el reconocimiento inicial y la práctica básica del sistema Braille.

Se incorpora una estructura básica compuesta por archivos HTML, CSS, JavaScript y un banco de datos en formato JSON.

El proyecto incluye letras del alfabeto español, vocales acentuadas, números, signos básicos de puntuación y palabras de uso cotidiano.

Se registra la autoría del desarrollo, organización, adaptación digital, estructura de consulta y presentación web a cargo de Gabriel Berrospi, con apoyo institucional del CREBE "Señor de los Milagros" - Ucayali.

## Incorporación de respaldo documental

Se actualiza el archivo `README.md` para incluir la carpeta `docs`, el archivo `LICENSE` y una explicación más clara sobre el respaldo pedagógico, autoral, institucional y organizativo del proyecto.

Se actualiza el archivo `LICENSE`, con aclaración de que la licencia corresponde al código del repositorio y no otorga automáticamente derechos sobre logos institucionales, documentos externos, imágenes, materiales educativos o recursos de terceros.

Se agregan documentos de respaldo en la carpeta `docs`:

- `alcance-pedagogico.md`
- `fuentes-y-creditos.md`
- `uso-permitido.md`
- `respaldo-institucional.md`
- `estructura-del-proyecto.md`
- `bitacora-de-cambios.md`

Cada mejora significativa debe anotarse con fecha, descripción breve y motivo del cambio.

Formato sugerido:

```text

Cambio realizado:
Motivo:

```

Este registro no reemplaza los commits de GitHub, pero ayuda a documentar el sentido pedagógico, autoral e institucional de cada modificación importante.
